markmap

约 247 字小于 1 分钟

2025-03-01

概述

主题支持在 文章中 嵌入由 markmap 。

该功能由 @vuepress/plugin-markdown-chart 提供支持。

配置

主题默认不启用该功能。

你需要在你的项目中安装 markmap-lib, markmap-toolbar and markmap-view:

pnpm
pnpm add markmap-lib markmap-toolbar markmap-view
yarn
yarn add markmap-lib markmap-toolbar markmap-view
npm
npm i markmap-lib markmap-toolbar markmap-view

然后在 .vuepress/config.ts 配置文件中，启用该功能：

.vuepress/config.ts
export default defineUserConfig({
  theme: plumeTheme({
    markdown: {
      markmap: true, 
    },
  })
})

注

以下文档 Fork 自 @vuepress/plugin-markdown-chart, 遵循 MIT 许可证。

语法

```markmap
<!-- 在这里放置内容 -->
```

支持通过 Frontmatter 语法进行配置。

案例

markmap

````markmap
---
markmap:
  colorFreezeLevel: 2
---

````

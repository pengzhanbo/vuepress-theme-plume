介绍

约 345 字大约 1 分钟

2024-03-06

主题内置的使用的插件，扩展了主题的众多功能，你可以在 plugins 配置中， 实现对内部使用的各个插件的自定义配置。

配置

所有主题内部使用的插件， 均在 plugins 字段中进行配置。

.vuepress/config.ts
import { 
defineUserConfig
 } from 'vuepress'
import { 
plumeTheme
 } from 'vuepress-theme-plume'

export default 
defineUserConfig
({
  
theme
: 
plumeTheme
({
    
plugins
: {
      // more options...
    }
  }),
})

插件列表

  • @vuepress/plugin-nprogress - 页面加载进度条
  • @vuepress/plugin-photo-swipe - 图片预览
  • @vuepress/plugin-reading-time - 文章阅读时间
  • @vuepress/plugin-watermark - 文章水印
  • @vuepress-plume/plugin-search - 本地搜索
  • @vuepress/plugin-docsearch - Algolia 文档搜索
  • @vuepress/plugin-copy-code - 代码复制
  • @vuepress/plugin-shiki - 代码高亮
  • @vuepress/plugin-comment - 文章评论
  • @vuepress/plugin-markdown-hint - Markdown 提示
  • @vuepress/plugin-markdown-image - Markdown 图片
  • @vuepress/plugin-markdown-math - Markdown 数学公式
  • @vuepress/plugin-markdown-include - Markdown 导入文件
  • @vuepress/plugin-markdown-chart - Markdown 图表 chartjs / echarts / mermaid / flowchart / markmap / plantUML
  • @vuepress/plugin-replace-assets - 资源链接替换
  • vuepress-plugin-md-power - Markdown Power
  • @vuepress/plugin-git - git 提交信息
  • @vuepress/plugin-cache - 页面编译缓存
  • @vuepress/plugin-seo - SEO 优化
  • @vuepress/plugin-sitemap - 站点地图

提示

您无需重复安装这些内置插件，也无需在 vuepress配置 > plugins 中添加它们。主题已在内部完成了这些工作。

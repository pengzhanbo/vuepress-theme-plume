概述

主题提供了 嵌入 AcFun 视频 的功能。

该功能由 vuepress-plugin-md-power 提供支持。

配置

该功能默认不启用。你需要在主题配置中开启。

.vuepress/config.ts
export default defineUserConfig({
  theme: plumeTheme({
    markdown: {
      acfun: true, 
    },
  })
})

语法

简单的语法：

@[acfun](id)

更多选项：

@[acfun width="100%" height="400px" ratio="16:9"](id)

选项说明：

  • id: 视频 ID
  • width: 视频宽度
  • height: 视频高度
  • ratio: 视频比例，默认 16:9

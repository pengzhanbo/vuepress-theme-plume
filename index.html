Theme Plume

VuePress Next Theme

一个简约易用的，功能丰富的 vuepress 文档&博客 主题

快速开始 →Github

响应式布局

适配移动设备，PC，平板

博客 & 文档

无论是想写博客，或想写产品文档，或者两者兼顾

开箱即用

零配置开箱即用，丰富的自定义配置

多语言

内置 中文、英文等 7 种语言支持，还可以自定义添加更多的语言支持

双色主题

支持 浅色/深色 主题，包括代码高亮

插件

内置丰富的插件，一站式解决网站一般需求

搜索、评论

支持多种评论系统，支持本地搜索、Algolia搜索

加密

支持全站加密、部分加密（加密目录、加密文章）

Markdown 增强

在常规 markdown 语法的基础上，添加数十种增强语法，如自定义容器、文件树、代码分组等

功能

内置丰富的功能，满足网站一般需求。

  • 文章信息

    为文章添加标签、分类、字数统计、阅读时间、写作日期等信息。

  • 评论

    支持 4 种评论系统：Giscus、Waline、Twikoo、Artalk
    你可以自由选择符合你的需求的评论系统。

  • 搜索

    支持基于 minisearch 的本地搜索，还支持接入 Algolia 搜索。

  • 加密

    支持全站加密、部分加密（加密目录、加密文章）。

  • 代码

    代码复制，CodePen演示，JSFiddle演示，CodeSandbox演示，代码组，行高亮，行聚焦，行警告，差异对比，代码块折叠等。

  • 资源嵌入

    图表：chart.js/ECharts/Mermaid/flowchart
    视频：Bilibili/Youtube/ArtPlayer
    PDF，200K+ Iconify 图标

博客

主题默认支持博客，生成你的个人博客。

  • 文章列表

    通过文章写作日期，自动排序并生成博客文章列表页。

  • 博主信息

    自定义名称、座右铭、头像，社交媒体链接。

  • 分类、标签、归档

    自动生成分类页、标签页，为文章根据年份进行归档。

文档

主题默认支持文档，生成你的产品文档，或归纳你的知识体系。

  • 侧边栏

    根据文档目录自动生成侧边栏，也可以手动配置，主题提供了更简单的配置方式。

  • 文档分类归纳

    可以文档的不同，归纳到不同的目录，更好的管理文档结构。

使用

使用以下命令，快速创建一个新的使用本主题的 VuePress 项目。

pnpm
pnpm create vuepress-theme-plume@latest
yarn
yarn create vuepress-theme-plume@latest
npm
npm create vuepress-theme-plume@latest

启动项目

pnpm
pnpm docs:dev
yarn
yarn docs:dev
npm
npm run docs:dev

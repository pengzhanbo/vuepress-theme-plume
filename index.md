### 使用

使用以下命令，快速创建一个新的使用本主题的 ==VuePress== 项目。

:::npm-to

```sh
npm create vuepress-theme-plume@latest
```

:::

### 启动项目

::: npm-to

```sh
npm run docs:dev
```

:::

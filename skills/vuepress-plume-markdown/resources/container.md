# Containers

Use `::: type [title]` to create alert boxes or collapsible details.

## Supported Types

- `note`, `info`, `tip`, `warning`, `caution` (Alerts)
- `details` (Collapsible)

## Syntax

```md
::: type [Optional Title]
Content
:::
```

## Examples

```md
::: tip
This is a tip.
:::

::: warning CAUTION
Custom title for warning.
:::

::: details Click to expand
Hidden content.
:::
```

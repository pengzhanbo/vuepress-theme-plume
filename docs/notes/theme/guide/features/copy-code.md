## 使用

主题默认启用了代码复制功能。支持 一键复制 在 文章中 展示的代码。

默认情况下， 主题会为每一个 代码块 添加一个复制按钮。该按钮仅在桌面端显示。

当鼠标悬停在 代码块 上时，在右上角会出现一个复制按钮。

## 效果

### 代码块

```js
const a = 1

export default a
```

### 代码组

::: code-tabs
@tab foo.js

```js
const a = 1

export default a
```

@tab bar.js

```js
const b = 1

export default b
```

:::

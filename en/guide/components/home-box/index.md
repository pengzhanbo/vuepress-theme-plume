## Home Layout Container

The `<HomeBox>` component provides a wrapper container for areas when customizing the homepage layout.

## Props

:::: field-group

::: field type
@type `string`
@default `''`
@optional

Area type.
:::

::: field full
@type `boolean`
@default `false`
@optional

Whether to enable fullscreen mode.
:::

::: field background-image
@type `string`
@default `''`
@optional

Area background image.
:::

::: field background-attachment
@type `'fixed' | 'local'`
@default `'local'`
@optional

Area background attachment method.
:::

::::
